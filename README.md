# dev-assist 🧰

Assistant CLI interactif pour créer, gérer et déployer des projets fullstack :

- NestJS
- React / Vite
- Next.js
- Spring Boot
- Symfony
- .NET Core
- Docker, Git, CI/CD
- Monorepo front+back

## 🚀 Installation

```bash
curl -fsSL https://raw.githubusercontent.com/tomtomysteria/dev-assist/main/install.sh | bash
```

## ❌ Désinstallation

```bash
bash uninstall.sh
```
